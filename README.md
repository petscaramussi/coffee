# Coffee
Um App de e-commerce feito pensado em uma cafeteria, web site - mobile first

## Instalação
Angular 
<br>
 `npm install`

.Net
<br>
 ` dotnet restore`

## Rodar o Projeto

Angular
<br>
 `npm start`
 > (centifique-se de estar dentro da pasta 'coffee/client')

.Net
<br>
 `dotnet watch`
 > (centifique-se de estar dentro do caminho 'coffee/API/api'
